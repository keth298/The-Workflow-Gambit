# LargeDataset — Chess Data for RAG Engine

This directory holds all source datasets used by the RAG chess engine.
After downloading, run `python ../rag_engine/data_pipeline.py` to build the
SQLite database, then `python ../rag_engine/build_index.py` to build the
FAISS vector index.

---

## Required datasets

### 1. Chess Games (PGN archive)
- **Source:** https://github.com/angeluriot/Chess_games
- **Target path:** `LargeDataset/chess_games/`
- **Format:** PGN files
- **Download:**
  ```bash
  git clone https://github.com/angeluriot/Chess_games chess_games
  ```

### 2. All Chess Openings (Kaggle)
- **Source:** https://www.kaggle.com/datasets/alexandrelemercier/all-chess-openings
- **Target path:** `LargeDataset/openings/`
- **Format:** CSV
- **Download (requires Kaggle API key):**
  ```bash
  kaggle datasets download -d alexandrelemercier/all-chess-openings -p openings --unzip
  ```
  Or download manually and place the CSV at `openings/chess_openings.csv`.

### 3. Lichess Puzzle Dataset (Kaggle)
- **Source:** https://www.kaggle.com/datasets/tianmin/lichess-chess-puzzle-dataset
- **Target path:** `LargeDataset/puzzles/`
- **Format:** CSV
- **Download:**
  ```bash
  kaggle datasets download -d tianmin/lichess-chess-puzzle-dataset -p puzzles --unzip
  ```
  Or download manually and place the CSV at `puzzles/lichess_puzzles.csv`.

## Expected directory layout after download

```
LargeDataset/
├── chess_games/
│   └── *.pgn          (one or more PGN files with master/amateur games)
├── openings/
│   └── *.csv          (opening ECO/name/moves CSV)
├── puzzles/
│   └── *.csv          (Lichess puzzle CSV)
├── chess_data.db      (generated by data_pipeline.py)
├── faiss.index        (generated by build_index.py)
└── id_to_fen.pkl      (generated by build_index.py)
```
